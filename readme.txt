WhoisMind.com widget allows users to enter a hostname, ip address or email and displays all public domain information and related data.

== Description ==

WhoisMind.com widget has the following features:

*   allows users to enter hostname, email address and ip address
*   redirects users to <a href="http://whoismind.com" title="Visit WhoisMind.com">WhoisMind.com</a> for public domain information and related data 

== Installation ==

1. Go to WP admin site to install plugin and activate
1. Access the widget from the Appearance -> Widgets screen in the Dashboard
1. Drag the widget from the Available Widgets or Inactive Widgets area on the left into the Sidebar area on the right.
1. When you see a dashed line appear, you can drop the widget into place.
1. To make changes to a widget in the sidebar, click on the triangle on the right side.
